Security Architecture

Built on open standards and browser-native security primitives.

  • AES-256-GCM encryption via Web Crypto API
  • PBKDF2 key derivation (100,000 iterations, SHA-256)
  • WebRTC DTLS/SRTP transport encryption
  • Peer-to-peer model, no media relayed through servers
  • Notes encrypted at rest in localStorage
  • SHA-256 consent integrity hashes
  • No third-party analytics or tracking
  • Open-source under OWASP BLT
  • Content Security Policy headers
  • Browser-only, no backend and no accounts
  • GDPR-ready consent export (JSON/CSV)
  • Voice noise suppression and echo cancellation
